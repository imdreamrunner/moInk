moInk
=====

This is the source codes for moink.me.  moInk is an online image designer using HTML5 and it is served by node.js.

Main Author
-----------

Zhou Xinzi (founder of moInk project)


Techniques
----------

### moInk Designer

+ jQuery library
+ underscore library
+ Front-end MVC using backbonejs
+ CSS Less

### node server

+ express
+ mongodb with mongoose
+ ejs
+ node-canvas
